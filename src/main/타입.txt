1. 타입
    1] 기본타입
    boolean char byte short int long float double

    2] 참조타입 (기본타입 8가지 외 모두)
    Class 타입 : String, ArrayList, Math, Dto, Dao 등등
    배열 타입 : String[], ArrayList[] 등등
    Interface 타입 : List, Map, mapper, Repository 등등
    열거 타입 : enum

2. 이름 정의
    자바에서 첫글자를 대문자로 쓰는 경우 : 클래스명, 인터페이스명, 상수명, 열거 등등
    자바에서 첫글자를 소문자로 쓰는 경우 : 변수명, 메소드명, 패키지명 등등

3. 타입, 변수, 변수값 차이
    타입 : 실제 값을 저장하려는 형태, 변수에 저장된 자료/데이터/리터럴/값의 종류/형태/형식
    변수 : 실제 값이 담기는 공간, 자료/데이터/리터럴/값을 저장할 수 있는 메모리 공간, 수정가능
    변수명 : 메모리 공간의 상징적인 이름
    변수값 : 실제 값, 메모리 공간의 저장된 값
